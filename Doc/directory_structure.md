# MQLのディレクトリ構造について

リポジトリMQLの各ディレクトリの概要を記載します。
これらのディレクトリは、プラットフォームのバージョンによって異なる可能性があります。

1. MQL4/5フォルダ  
MQL4/5言語で書かれたプログラムファイルが格納されます。  
このフォルダは、MetaTrader 4/5インストールフォルダ内にあります。
当該リポジトリは、このディレクトリを基に作られています。

# MQL4とMQL5のディレクトリ構成

## MQL4のディレクトリ構成

```
MQL4
├─Experts
├─Files :EAによって生成されたファイルを保存する
├─Images :カスタム画像を保存する
├─Include
│  ├─Arrays: 配列に関する機能
│  ├─Canvas: カスタムチャート描画用の機能
│  ├─ChartObjects : テキスト、ライン、矢印などのチャートオブジェクトに関する機能
│  ├─Charts : チャートの情報や、チャートに関する操作などの、チャートに関する機能
│  ├─Controls : ボタン、ラジオボタン、チェックボックスなどのUI要素を作成・変更するといった、カスタムコントロールに関する機能
│  │  └─res
│  ├─Files : EAやスクリプトが出力するファイルを保存するためのフォルダー
│  ├─Indicators : カスタムインジケーターに関するファイルを保存する
│  ├─Strings : 文字列操作に関する機能
│  └─Tools : 一般的なユーティリティ機能
├─Indicators : 標準のテクニカルインジケーターファイルが保存される
│  └─Examples : 標準インジケーターの例
│      └─SimplePanel : 簡単なパネルを持つ標準インジケーターの例
├─Libraries : カスタムライブラリファイルが格納できる
├─Logs : ログファイルが格納される
├─Presets : テンプレートファイルが格納できる
└──Scripts :　スクリプトファイルが格納できる
   └─Examples : スクリプトの例
       ├─DLL : DLLファイルに関する例
       └─Pipes : パイプに関する例
```

## MQL5のディレクトリ構成

```
MQL5
├─Experts : EAを格納するフォルダー
│  ├─Advisors : 自動売買を実行するEAが含まれる
│  └─Examples : EAのサンプル
│      ├─ChartInChart           : チャート内の別のチャートを表示する方法を示す例
│      ├─Controls               : グラフィカルユーザーインターフェイス(GUI)を作成するためのカスタムコントロールの例
│      ├─Correlation Matrix 3D  : 3D相関マトリックスの作成方法を示す例
│      ├─MACD                   : チャート上にMACDインジケーターを表示する方法を示す例
│      ├─Math 3D                : 3D図形やテクスチャを作成するための数学的アルゴリズムの例
│      │  └─Sets                : 数学的セット理論に関連する例
│      ├─Math 3D Morpher        :  3D形状を変形させる方法を示す例
│      │  └─Textures            : 3Dテクスチャ変形の例
│      └─Moving Average         : 移動平均を計算する方法を示す例
├─Files : EAやスクリプトが出力するファイルを保存するためのフォルダー
├─Images :  カスタムインジケーターで使用されるグラフィックファイルを格納するためのフォルダー
├─Include : 外部ライブラリ、カスタムインジケーター、およびEAで使用するカスタムインクルードファイルを格納する
│  ├─Arrays         : 配列に関する機能
│  ├─Canvas         : カスタムチャート描画用の機能
│  │  ├─Charts          : カスタムチャート描画に使用できる機能
│  │  └─DX              : Direct3Dに関する機能。DirectXを使用して3D描画を行うことができる
│  │      └─Shaders         : シェーダーファイルが格納されている
│  ├─ChartObjects   : テキスト、ライン、矢印などのチャートオブジェクトに関する機能
│  ├─Charts         : チャートの情報や、チャートに関する操作などの、チャートに関する機能
│  ├─Controls       : ボタン、ラジオボタン、チェックボックスなどのUI要素を作成・変更するといった、カスタムコントロールに関する機能
│  │  └─res             : カスタムコントロールに使用される画像やアイコンなどのリソースを格納できる
│  ├─Expert         : EAの取引情報や、ポジションを管理する機能、口座情報を取得するといった、EAに関する機能
│  │  ├─Money           : 口座残高や証拠金維持率を取得するといった金銭管理に関する機能
│  │  ├─Signal          : シグナルの生成や、シグナルの履歴を取得するといった、シグナルに関する機能
│  │  └─Trailing        : トレイリングストップに関する機能
│  ├─Files          : ファイル操作に関する機能
│  ├─Generic        : 汎用的な機能を提供するヘッダーファイルが格納されている
│  │  ├─Interfaces      : インターフェースの定義が格納されている
│  │  └─Internal        : 内部で使用されるヘッダーファイルが格納されている
│  ├─Graphics       : グラフィックスに関する機能
│  ├─Indicators     : インジケーターに関する機能
│  ├─Math           : 数学の機能に関する機能
│  │  ├─Alglib          : 数学のアルゴリズムを実装するためのライブラリであるAlglibのヘッダーファイルが格納されている
│  │  ├─Fuzzy           : ファジー論理に関する機能
│  │  └─Stat            : 統計に関する機能
│  ├─OpenCL         : OpenCLに関する機能
│  ├─Strings        : 文字列操作に関する機能
│  ├─Tools          : 一般的なユーティリティ機能
│  ├─Trade          : 取引に関する機能に関する機能
│  └─WinAPI         : Windows APIに関する機能
├─Indicators : 作成したカスタムインジケーターを格納する
│  └─Examples                : Indicatorsディレクトリ内のテクニカルインジケーターの使用例
│      ├─Canvas                 : Canvasを使用したグラフィック描画の例
│      └─Panels                 : パネルを使用した例
│          ├─ChartPanel             : ChartPanelを使用したパネルの例
│          └─SimplePanel            : SimplePanelを使用したパネルの例
├─Libraries : 共有ライブラリを格納する
├─Logs : ログファイルを保存する
├─Profiles : MetaTrader 5のプロファイルを保存するためのフォルダー
│  ├─Charts     : プラットフォームのチャートの設定が格納されている。各通貨ペアに対応するディレクトリがあり、それぞれの通貨のチャートの設定やカスタムインディケーター、テンプレートなどが格納される。
│  ├─Community  : ストラテジーテスターの最適化の履歴やデータ、EAの最適化レポートといった、MQL5コミュニティでのサービスとして提供される機能に関連するファイルが格納されている
│  ├─SymbolSets : シンボルのグループ化に関するファイルが格納されている。同じ通貨ペアの複数のシンボルを一度に表示したり、複数の通貨ペアを一度に表示する
│  ├─Templates  : チャートテンプレートに関するファイルを格納する。チャートの外観を一括して設定できる
│  └─Tester     : テスト結果、最適化設定、ストラテジーの設定といった、ストラテジーテスターに関するファイルを保存するために使用する
├─Scripts : スクリプトを格納するフォルダー
│  ├─Examples           : スクリプトの使用例
│  │  ├─AccountInfo         : アカウント情報を取得するスクリプトのサンプル
│  │  ├─ArrayDouble         : double型の配列を操作するためのサンプル
│  │  ├─Canvas              : カスタム描画オブジェクトを作成するためのサンプル
│  │  │  └─Charts           : チャート上にカスタム描画オブジェクトを表示するサンプル
│  │  ├─ObjectChart         : カスタムチャートオブジェクトを作成するサンプル
│  │  ├─ObjectSphere        : 3Dオブジェクトを作成するサンプル
│  │  ├─OpenCL              : OpenCLを使用して計算するサンプル
│  │  │  ├─Double           : double型のデータを使用してOpenCLを使用するサンプル
│  │  │  │  └─Kernels       : Kernels: カーネル関数を定義するためのサンプル
│  │  │  ├─Float            : float型のデータを使用してOpenCLを使用するサンプル
│  │  │  │  └─Kernels       : カーネル関数を定義するためのサンプル
│  │  │  └─Seascape         : シーンを描画するためのサンプル
│  │  ├─OrderInfo           : 注文情報を取得するためのサンプル
│  │  ├─PositionInfo        : ポジション情報を取得するためのサンプル
│  │  ├─Remnant 3D          : 3Dオブジェクトを作成するためのサンプル
│  │  │  └─Shaders          : シェーダーコードを定義するためのサンプル
│  │  └─SymbolInfo          : シンボル情報を取得するためのサンプル
│  └─UnitTests          : MQL5のユニットテストを作成するためのディレクトリ
│      ├─Alglib             : アルゴリズムライブラリに関するテストコード
│      ├─Fuzzy              : ファジィロジックに関するテストコード
│      ├─Generic            : 様々な関数、構造体、クラス、およびその他のMQL5の一般的な機能のテストコード
│      └─Stat               : 統計学関連のテストコード
└─Services : MT5サービスファイルを格納するためのフォルダー
```
